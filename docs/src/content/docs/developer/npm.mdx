Integrate capo.js into your project by installing it from [npm](https://www.npmjs.com/package/@rviscomi/capo.js).

## Installation and usage

```sh
npm install @rviscomi/capo.js
```

To use capo.js in your project, import it:

```js
import * as capo from '@rviscomi/capo.js';
```

### `colors`

The `colors` library is used for accessing and generating the color pallettes that capo.js uses to render the visualization.

#### Properties

##### `DEFAULT`

The default capo.js color pallette:

#### Functions

##### `generateSwatches`

Given a parameter `hue`, (e.g. 320 for pink and 200 for blue), generate an array of 11 colors that range from dark to light, followed by grey (`#ccc`).

```js
const pinkPallette = capo.colors.generateSwatches(320);
```

##### `getInvalidBackgroundColor`

Given a parameter `elementColor`, generate the CSS style for a `repeating-linear-gradient` that alternates between `elementColor` and `#ccc`.

```js
const invalidBackgroundColor = capo.colors.getInvalidBackgroundColor('#9e0142');
```
